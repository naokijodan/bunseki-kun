# コード一貫性チェック

以下の観点でコードの一貫性をチェックしてください：

## 1. ストレージアクセス
- `chrome.storage.local` と `chrome.storage.sync` が混在していないか
- 保存と読み込みで同じストレージを使っているか

## 2. キー名の一致
- 保存時のキー名と読み込み時のキー名が一致しているか
- 例: `claudeApiKey` で保存して `claudeAPIKey` で読み込んでいないか

## 3. API呼び出し
- 同じAPIに対して異なる呼び出し方法を使っていないか
- エラーハンドリングのパターンが統一されているか

## 4. ファイル間の整合性
- options.js で保存したものを popup.js で正しく読み込めるか
- background.js と他ファイルでメッセージ形式が一致しているか

問題があれば具体的な箇所と修正案を提示してください。
